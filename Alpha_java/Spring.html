Spring Framework

From Zero to Hero - Complete Guide

What is Spring Framework?

Spring is an open-source Java framework for building enterprise applications. It provides comprehensive infrastructure support for developing robust, maintainable Java applications with ease.

Key Points:

  • Lightweight and powerful
  • Promotes loose coupling through Dependency Injection (DI)
  • Follows SOLID principles
  • Production-ready and widely used in industry

Core Components

  • Spring Core: IoC Container and Dependency Injection
  • Spring MVC: Web framework for building web applications
  • Spring Data: Data access layer abstraction
  • Spring Security: Authentication and authorization
  • Spring Boot: Rapid application development

Why Use Spring?

  • Simplifies Java development
  • Reduces boilerplate code
  • Improves code testability
  • Provides declarative programming through annotations
  • Supports microservices architecture
  • Excellent community support

Getting Started - Basic Example

Here's a simple Spring Boot application:

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class MySpringApp {
    public static void main(String[] args) {
        SpringApplication.run(MySpringApp.class, args);
    }
}

Dependency Injection Example

import org.springframework.stereotype.Service;

@Service
public class UserService {
    public void createUser(String name) {
        System.out.println("User created: " + name);
    }
}

// Usage in another class
import org.springframework.beans.factory.annotation.Autowired;

@Component
public class UserController {
    @Autowired
    private UserService userService;
    
    public void handleRequest(String userName) {
        userService.createUser(userName);
    }
}

Common Annotations

  • @SpringBootApplication - Main entry point
  • @Service - Business logic layer
  • @Repository - Data access layer
  • @Controller - Web request handler
  • @Autowired - Dependency injection
  • @Component - Generic Spring-managed component
  • @Bean - Custom bean definition

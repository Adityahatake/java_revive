# Java Revive Roadmap

Welcome to the **Java Revive** repository! This project serves as a comprehensive path for mastering Java programming and data structures, starting from basic concepts to advanced problem-solving techniques.

## 📂 Repository Structure

The repository is organized to follow a step-by-step learning progression:

### 1. `Alpha_java` (Core Learning Path)
This directory acts as the central course material, divided logically into ordered components:
- `01_flowcharts` to `06_ternary_Operator_&_Switch_Case`: The fundamental building blocks of programming in Java.
- `07_loops` & `09_functions`: Control flow and reusability.
- `11_arrays`, `13_2D_Arrays`, and `14_strings`: Foundational data structures.
- `16_oops_Concept`: Object-Oriented Programming (OOP) principles.
- Data structures and algorithms deeper dive: `17_recursions`, `20_backtracking`, `21_linked_List`, `22_STACK`, `23_QUEUE`.
- Additional resources covering Spring and REST APIs.

### 2. `Week_1` (Alternative Learning Path)
Contains another collection of Java files focusing on early concepts, arrays, strings, oops concepts, and general problem-solving. Great for revisiting basics.

### 3. `Leetcode` (Problem Solving)
A consolidated directory containing solutions for various LeetCode problems (e.g., Two Sum, Merge Intervals, Palindromes). All previously scattered problems (such as those from `Java_Problems`) have been merged here for a unified structure.

## ⚙️ Setup and Execution

To run the codes in your local environment, you will need the Java Development Kit (JDK) installed.
1. Navigate to the directory containing your `.java` file.
2. Compile the code using `javac`:
   ```sh
   javac Hello.java
   ```
3. Run the compiled class file:
   ```sh
   java Hello
   ```

*Note: All compiled `*.class` files are ignored by Git to keep this repository clean and lightweight.*
